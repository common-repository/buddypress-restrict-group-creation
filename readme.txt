Extend restricting group creation with mappings to WordPress Capabilities and various thresholds

== Description ==

This plugin extends the default "Restrict group creation to Site Admins?" settings within BuddyPress. Rules for restricting group creation can be attached to each wp capability giving multiple levels of control. The rules are made up of member thresholds: forum posts, friends, status updates, days since registered, total groups created, total group admin memberships, total group memberships and total group mod memberships; Also supports the third-party plugin: achievements. S for example you can restrict group creation to members who have more than 10 friends etc.

This plugin was originally created by Rich @ etivite, and has been updated to run on the latest version of WordPress by Venutius.

== Installation ==

1. Upload the full directory into your wp-content/plugins directory
2. Activate the plugin at the plugin administration page
3. Open the plugin configuration page, which is located under BuddyPress and map the wp-capabilities to overall group creation, forum option, and group status level.
